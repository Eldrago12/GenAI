# GenAI HUB

## Overview
This GenAI HUB is designed to provide various AI-powered features through a user-friendly web interface. It leverages cutting-edge technologies to offer functionalities like code generation, notes summarization, chatbot interaction, and image generation.

## Features
1. **Code Generation**: Utilizes AWS Bedrock for code generation, enabling users to automate the process of writing code snippets for various programming tasks.

2. **Notes Summarizer**: Integrates the Cohere Foundation model powered by AWS Bedrock to summarize lengthy texts efficiently, aiding in research and content analysis.

3. **Chatbot**: Incorporates the Llama and Langchain models for chatbot interaction. The frontend is built using Streamlit, providing a seamless conversational experience for users.

4. **Image Generation**: Implements the Stable Diffusion model for image generation, allowing users to create high-quality images based on their specifications.
